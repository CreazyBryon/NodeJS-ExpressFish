Developed on Visual Studio 2008

Command Line Sample:

1. Add an environment variable

HTS.EnvironmentVariableUtility.exe /EnvVarName:"Path" /EnvVarValue:"E:\EBI R430\Server\WorkTimeMonitor\"

2. Remove an environment variable

HTS.EnvironmentVariableUtility.exe /u /EnvVarName:"Path" /EnvVarValue:"E:\EBI R430\Server\WorkTimeMonitor\"
